In today's session we leant a little bit more about Github.

The advantages of using Github are many.

It allows you to:

  • Create branches to develop features
  • Identify when bugs are introduced to your code
  • Revert back to previous versions of the code
  • Have multiple developers working on separate features simultaneously
  • Update only the parts of the code that you have changed so others' work isn't affected
  • Monitor the development and manage the project
